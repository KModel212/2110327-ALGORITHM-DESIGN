# 🧠 Algorithm Design (2110327)

This repository contains a collection of programming solutions and algorithm implementations developed for the **Algorithm Design (2110327)** course using c++.

The goal of the course is to study fundamental algorithmic techniques and apply them to solve computational problems efficiently.

## 📚 Course Overview

The assignments focus on designing algorithms that handle computational challenges efficiently in terms of **time complexity** and **space complexity**.

Students are required to analyze problems, select appropriate algorithmic strategies, and implement solutions that pass automated test cases.
